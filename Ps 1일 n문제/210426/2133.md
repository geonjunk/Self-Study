# 문제 : 2133번 - 타일 채우기
* 링크 : https://www.acmicpc.net/problem/2133
* 타일채우기 문제의 어려운 난이도 문제

## 풀이
* 나열을 통해 점화식을 유도하는 DP문제
* 우선 n은 짝수일때만 타일로 채울 수 있음
* n=2부터 나열해보면
    * n=2일때는 3가지 모양나옴 ->dp[2]=3;
    * n=4일때는 이전 경우인 n=2일때에서 3 * 2타일을 새로 붙히는 경우의 수이기 때문에 dp[2]*3인데 여기에 추가로 3 * 4 타일을 한번에 새로운 방법으로 채우는 방법 2가지도 존재하므로 -> dp[4]= dp[2] * 3 + 2
    * n=6일때도 똑같이 dp[4]*2에다가 3 * 4를 새롭게 채우는 방법, 3 * 6을 새롭게 채우는 방법을 더해야하기 때문에 - >  dp[6] = dp[4] * 3+dp[2] * 2+ 2
    * 사진

* 이를 통해 점화식 유도하면
    * n>=4 : dp[n]=dp[n-2]*3+dp[n-4]*2+....+2
    * n=2 : dp[2]=3

* 코드)
    ```
    #include<iostream>
    using namespace std;

    int dp[31];
    int n;
    int main() {
        cin >> n;
        dp[2] = 3;
        dp[0] = 1;
        for (int i = 4; i <= n; i++) {
            if (i % 2 == 0) {
                dp[i] = dp[i - 2] * dp[2];
                for (int j = i - 4; j >= 0; j -= 2) {
                    dp[i] += dp[j] * 2;
                }
        
            }
        }
        cout << dp[n] << "\n";

        return 0;
    }
    ```
